## Hubotとは？

### 概要

- node.jsでbotを動かすためのフレームワーク
- npmで拡張モジュールの導入が簡単
- Adapterモジュールで様々なチャットツールに対応
- CoffeeScriptかJavaScriptで機能追加

### できること

- 定期的な告知
- コマンドの実行結果を通知
- スクリプトで書ければだいたいできる

## 活用例

- AWSにEC2のアドレスを問い合わせて通知
- 天気予報サイトから情報取得して通知
- 決まった単語に反応
